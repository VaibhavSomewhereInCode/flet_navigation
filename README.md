# Flet Navigation

A simple and reusable navigation system for Flet apps.

I originally made this because the default Flet navigation/routing system in Flet felt unnecessarily complicated for small and medium projects.  
So instead of rewriting navigation logic every time, I created a generalized structure that anyone can download, modify, and use in their own apps.

---

# Features

- Simple navigation system
- Beginner friendly
- Reusable structure
- Easy page management
- Clean organization
- Multi-file support

# Why This Exists

Flet's default navigation system can become confusing for beginners because of:
- routes
- views
- callbacks
- version changes
- API updates

This project focuses on simplicity and reusability instead.

---

# Goal

The goal of this project is to provide:
- an easy starting template
- understandable navigation logic
- reusable navigation components
- a cleaner learning experience for beginners
